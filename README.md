# Classic Snake
:)Snake Game - Christmas Edition

A festive take on the classic Snake game, created using Python's turtle graphics library. In this game, you control a snake that grows longer as it eats food while avoiding collisions with the walls. The game is wrapped in a cheerful Christmas theme to bring some holiday spirit to the gameplay!

:)Features

Classic Snake Gameplay: Control the snake using the arrow keys to eat food, grow longer, and avoid running into walls.

Christmas Theme: Enjoy a festive holiday theme, complete with a white background, Christmas-themed candy, and a santa sprite.

Simple Controls: Use the arrow keys (Up, Down, Left, Right) to control the direction of the snake.

Score Tracking: Your score increases each time the snake eats food, with the length of the snake growing accordingly.

Fun Sounds & Graphics: Christmas-themed graphics to keep the game joyful.

:)How to Play

Start the Game: Simply run the game script, and you’ll be greeted with a Christmas-themed Snake game window.

Control the Snake: Use the arrow keys to guide the snake towards the food. Each time the snake eats a piece of food, it will grow longer.

Avoid Collisions: Don’t let the snake collide with the walls or the game will restart!

Christmas Food: The food is specially designed to match the Christmas theme, so you’ll be collecting festive items!

Enjoy the Festivities: The game colors have a Christmas twist to make it more fun and festive.

:)Game Controls

Up Arrow: Move the snake up.

Down Arrow: Move the snake down.

Left Arrow: Move the snake left.

Right Arrow: Move the snake right.
